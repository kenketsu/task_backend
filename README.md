# ともすすタスク管理

友人と使う想定で作ったタスク進捗管理アプリケーションです。<br>
使用者相互のタスクの進捗、作業時間などが確認できます。<br>
ログイン後はシングルページアプリケーションとなっています。<br>
フロントエンドはReactとRedux ToolKitで制作し、Firebase Hostingを用いて公開しています。
